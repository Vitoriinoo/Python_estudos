# Exercícios Consolidados: Lógica de Programação em Python

Este arquivo é um compilado de exercícios práticos que desenvolvi para revisar e consolidar os fundamentos da linguagem Python. 

Como venho estudando diversos conceitos sequenciais, decidi agrupar os principais desafios em um único script estruturado. Isso me ajudou a ter uma visão geral de como diferentes estruturas lógicas interagem entre si em cenários variados.

## Conceitos Praticados neste Arquivo

O código contém 10 blocos de exercícios distintos, onde coloquei em prática as seguintes habilidades:

- Controle de Fluxo: Uso de if, elif e else para tomada de decisões lógicas (como verificar maioridade ou comparar números).
- Estruturas de Repetição (Loops): Aplicação de laços while para manter programas rodando até que uma condição seja satisfeita (como sistemas de login e validação contínua de inputs) e laços for para iteração com limites definidos.
- Manipulação de Listas: Uso do método .append() para salvar dados inseridos pelo usuário, além da aplicação de funções nativas como sum(), max(), min() e len() para extrair métricas de forma limpa.
- Matrizes (Listas de Listas): Construção de estruturas bidimensionais para armazenar dados complexos, iterando sobre elas com laços for aninhados (utilizado no exercício de médias de alunos e provas).
- Validação de Dados: Criação de lógicas que impedem o usuário de inserir dados inválidos (como garantir que uma nota registrada esteja estritamente entre 0 e 10).
